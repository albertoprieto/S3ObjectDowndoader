# Amazon S3 cfditoday Descargador

Este script en Python utiliza la biblioteca PySimpleGUI y boto3 para descargar archivos desde Amazon S3 en función de un rango y tipo especificados.

## Requisitos

- Python 3.x
- Bibliotecas: boto3, PySimpleGUI

Instala las dependencias usando:

```bash
pip install boto3 PySimpleGUI
```

Ó instala las dependencias desde el requirements usando:
```bash
pip install -r requirements.txt
```
